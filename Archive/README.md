# Archive

The single home for retired content: shipped or abandoned releases, closed risks, superseded
decisions, descoped requirements. Files move here mirroring their original path
(`Archive/Decisions/…`, `Archive/Modules/…`, `Archive/Roadmap/…`); nothing is deleted, so links
survive. **Don't load this folder unless the task is explicitly about history.**
Rules: [`.claude/rules/archive.md`](../.claude/rules/archive.md).
