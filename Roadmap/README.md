## How to read this

**Horizons**
- **Now** — in build or launching this quarter.
- **Next** — committed for the next release wave.
- **Later** — direction and bets. Not yet scoped or committed. Detail in [Vision.md](Vision.md).

**Every item follows one shape:** one-line what-it-is · value / who it's for · status + horizon ·
scope confidence / key risk · links (spec, epic, feedback, decisions). Per-initiative detail lives
in the `<Initiative>.md` files linked below.
