# Writing in the brain

- Markdown; one idea per file. Write tight and high-signal — the agent reads these, every token counts.
- File names: `kebab-case.md`. Module/capability folders: `Title-Case-Kebab`.
- Decisions are numbered `ADR-00X`. Requirement spec files are named for the requirement; record the
  Jira key inside the file, not in the filename.
- Link related docs with relative paths; link liberally.
