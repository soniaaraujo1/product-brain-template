# Competitive Intel

Our knowledge about the market and competitors. Produced and maintained by
[`/competitor-analysis`](../.claude/skills/competitor-analysis/SKILL.md).

## How this folder is organized

- **One folder per competitor** (`Title-Case`), holding short folder-relative files:
  - `README.md` — the profile / teardown (start here)
  - `battlecard.md` — the GTM one-pager
  - `roadmap-comparison.md` (or `<topic>-comparison.md`) — focused deep-dives
  - `resources/` — that competitor's binaries (decks, screenshots, source PDFs/images)
- **`resources/`** (this level) — shared, cross-competitor material (market reports, market maps, general screenshots).
- **`market-<topic>.md`** — market scans not tied to a single competitor.

Conventions: [`.claude/rules/competitive-intel.md`](../.claude/rules/competitive-intel.md). Method + frameworks: [`.claude/skills/competitor-analysis/`](../.claude/skills/competitor-analysis/SKILL.md).
